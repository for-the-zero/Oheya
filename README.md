## 中文

一个基于大模型的知识图谱搜索引擎：输入任意关键词，AI 自动搜索网页、提取关键信息，输出结构化知识图谱（实体、属性、标签、图表）

对Magi搜索的一次利用大模型的模仿与进化

### 功能

- **AI 驱动搜索** — 接入多种搜索引擎 API（Exa、Tavily、Querit、UApi）
- **知识图谱输出** — AI 自动分析搜索结果，提取实体、属性、标签、关联内容，支持图表
- **流式输出** — 实时展示 AI 推理和生成过程
- **PWA 支持** — 可安装到桌面
- **自定义 AI 配置** — 支持任意 OpenAI 兼容 API
- **多重展示形式** — 包含概述、描述、属性、标签、图表、相关内容等内容

### 技术栈

[Vite](https://vite.dev/) + [Vue 3](https://vuejs.org/) + [PrimeVue](https://primevue.org/) + [TypeScript](http://typescriptlang.org/) + [Tailwind CSS](https://tailwindcss.com/) + [AI SDK](https://sdk.vercel.ai/) + [Pinia](https://pinia.vuejs.org/)

# EN

An AI-powered knowledge graph search engine built as a spiritual successor to Magi. Enter any keyword, AI searches the web, extracts key info, and outputs structured knowledge graphs (entities, attributes, tags, charts) alongside classic search results.

### Features

- **AI-Driven Search** — pluggable search backends (Exa, Tavily, Querit, UApi)
- **Knowledge Graph Output** — AI analyzes results into entities, attributes, tags, related content, charts
- **Streaming Output** — real-time AI reasoning & generation display
- **PWA** — installable, works offline
- **Custom AI Config** — any OpenAI-compatible API
- **Rich Presentation** — overviews, descriptions, attributes, tags, charts, related content

### Tech Stack

[Vite](https://vite.dev/) + [Vue 3](https://vuejs.org/) + [PrimeVue](https://primevue.org/) + [TypeScript](http://typescriptlang.org/) + [Tailwind CSS](https://tailwindcss.com/) + [AI SDK](https://sdk.vercel.ai/) + [Pinia](https://pinia.vuejs.org/)

### Quick Start

```bash
npm install
npm run dev
npm run build
```
